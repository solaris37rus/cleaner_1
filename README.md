# ЧистоИваново — сайт клининга для GitHub Pages

Готовый статический сайт семейного клининга в Иваново: услуги, прайс, калькулятор предварительной стоимости, корзина выбранных услуг и форма заявки.

Проект специально адаптирован под бесплатный GitHub Pages. Backend и секретные Telegram-токены не используются. Сайт формирует готовый текст заявки, копирует его в буфер обмена и даёт клиенту кнопки для отправки в Telegram или VK.

## Контакты

- Город: Иваново

## Почему без backend

GitHub Pages размещает статические сайты. Поэтому автоматическая отправка в Telegram через Bot API невозможна без отдельного сервера, потому что токен бота нельзя хранить в браузере. В этой версии реализован безопасный бесплатный вариант:

1. Клиент выбирает услуги в калькуляторе.
2. Заполняет ФИО, телефон и адрес.
3. Нажимает «Скопировать заявку».
4. Открывает Telegram или VK и вставляет готовый текст заявки.

## Деплой на GitHub Pages через Actions

В проект уже добавлен workflow:

```txt
.github/workflows/deploy-pages.yml
```

Он автоматически собирает сайт и публикует папку `dist` на GitHub Pages при каждом push в ветку `main`.

### Настройки в GitHub

1. Открой репозиторий `solaris37rus/cleaner`.
2. Перейди в `Settings`.
3. Открой раздел `Pages`.
4. В блоке `Build and deployment` выбери `Source: GitHub Actions`.
5. Сохрани настройки, если GitHub попросит.
6. Перейди в раздел `Actions`.
7. Запусти workflow `Deploy to GitHub Pages` вручную или сделай push в `main`.
8. После успешного выполнения сайт будет доступен по адресу вида:

```txt
https://solaris37rus.github.io/cleaner/
```

## Что важно при загрузке файлов

Файлы проекта должны лежать прямо в корне репозитория:

```txt
package.json
index.html
src/
public/
.github/
vite.config.ts
```

Не нужно загружать папку `cleaner-github-pages-ready` внутрь репозитория целиком. Нужно открыть эту папку и загрузить её содержимое.

## Проверка после деплоя

1. Открой опубликованный сайт.
2. Нажми «Рассчитать стоимость».
3. Выбери услугу и дополнительные позиции.
4. Заполни форму.
5. Нажми «Скопировать заявку».
6. Открой Telegram или VK.
7. Вставь текст заявки и отправь его.

## Персональные данные

Сайт не сохраняет данные клиентов на сервере, потому что серверной части нет. Данные остаются в браузере клиента и попадают к вам только после ручной отправки сообщения в Telegram или VK.
